# Hello World

欢迎来到我的博客！这是我的第一篇文章。

## 关于这个网站

这个网站采用了复古的 **DOS 终端风格**设计，灵感来自于早期的命令行界面。我使用现代的技术栈来构建这个怀旧风格的网站：

- **React 18** - 用户界面框架
- **TypeScript** - 类型安全
- **Vite** - 快速的构建工具
- **GitHub Pages** - 静态网站托管

## 为什么选择这种风格？

在当今充满华丽动画和复杂交互的 Web 世界中，我想创造一个**简洁而独特**的体验：

- **专注内容** - 没有分散注意力的元素
- **快速加载** - 极简的设计意味着更快的加载速度
- **怀旧情怀** - 向计算机早期时代致敬

## 如何使用这个网站

你可以通过两种方式浏览：

1. **命令行方式** - 直接输入命令，如 `/blog`、`/about`
2. **菜单导航** - 点击顶部的菜单按钮
