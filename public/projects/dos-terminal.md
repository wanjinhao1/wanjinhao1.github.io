# DOS Terminal Homepage

A retro-style personal homepage with command-line interaction, inspired by early DOS interfaces.

## Features

- **Command-line interface**: Navigate the site using terminal-style commands
- **Retro aesthetics**: CRT scanlines, green phosphor text, pixel-perfect fonts
- **Responsive design**: Works on desktop and mobile devices
- **Blog system**: Markdown-based blog with Netscape Navigator-style browser modal
- **Guestbook**: Sign the guestbook with localStorage persistence

## Tech Stack

- **React 18** - UI framework with hooks
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **GitHub Pages** - Static site hosting
- **React Markdown** - Markdown rendering for blog posts

## Commands

- `/` - Clear screen and show welcome
- `/about` - About me
- `/blog` - View blog list
- `/blog [id]` - View specific blog post
- `/projects` - View projects list
- `/project [id]` - View project details
- `/skills` - Skills stack
- `/contact` - Contact information
- `/help` - Show help
- `/clear` - Clear screen
- `su <name>` - Sign guestbook
